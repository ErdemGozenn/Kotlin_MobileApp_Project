# Sahibinden Clone - Mobile Application Project

This project is a simple e-commerce (housing sales) application developed for the Android platform using Java. It allows users to list houses based on their budget, add them to a cart, and simulate a purchase process. The application demonstrates the use of fundamental Android components, data transfer between Activities, and basic local database implementation.

---

## üåü Features

* **Login Screen:** A welcome screen where the user starts the application with a specific budget.
* **Home Page (Listing):** Lists properties by city (Ankara, Istanbul, Izmir, Adana) and allows filtering via a Spinner.
* **Purchase Process:** Users can add a selected property to their cart using the "Buy" (Satƒ±n Al) button, confirming the action with an AlertDialog.
* **Cart Screen:** Displays the properties added to the cart, their prices, total spending, and the user's remaining budget.
* **Budget Tracking:** The remaining budget is updated and displayed as purchases are made.
* **Multi-Language Support:** Ability to switch between Turkish (TR) and English (EN).
* **Brightness Control:** Adjust the device's screen brightness from within the app using a SeekBar.
* **Database:** City and property information is retrieved from a local SQLite database.
